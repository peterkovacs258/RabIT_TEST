The application was tested in XAMPP's APACHE webserver, and MYSQL database.
The sql export can be found inside the sql_export folder.



IMPORTANT ///////////////////
The project only works if the name of the main folder is 'rabittest'.
After downloading it from github the name will be RabIT_Test-master,
you must rename this folder before accessing the files.
/////////////////////////////
access url: localhost/rabittest

This application is a small php project, using MVC desing model.
It displays the users, and their recomended advertisements (the content of the database) in different pages,
and allows us to add more users to the database.
